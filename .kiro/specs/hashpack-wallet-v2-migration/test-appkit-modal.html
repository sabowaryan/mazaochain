AppKit Modal Opening Test

Prerequisites

  • Set NEXT_PUBLIC_USE_APPKIT=true in .env.local
  • Set valid NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID
  • Install HashPack wallet extension
  • Restart the development server

Test Instructions

This is a manual test guide. Follow these steps in your actual application:

Step 1: Navigate to a page with AppKitButton

Go to any page that uses the WalletConnectionWrapper or AppKitButton component.

Step 2: Check Button Rendering

  • Button should display "Initializing..." briefly
  • Button should then show the AppKit connect button
  • No error messages should appear

Step 3: Click the Button

Step 4: Verify Modal Opens

  • AppKit modal should appear on screen
  • Modal should show wallet connection options
  • HashPack should be listed as an option
  • Modal should be responsive and properly styled

Step 5: Test Modal Interactions

  • Click outside modal - should close
  • Click X button - should close
  • Select HashPack - should trigger connection flow

Expected Behavior

✓ Success Criteria:
  • Button click opens the AppKit modal immediately
  • Modal displays wallet connection options
  • No console errors appear
  • Modal is properly styled and functional

Troubleshooting

Modal doesn't open?

  • Check browser console for errors
  • Verify NEXT_PUBLIC_USE_APPKIT=true
  • Verify WalletConnect Project ID is valid
  • Check that AppKit initialized successfully

Button shows error?

  • Check the error message displayed
  • Verify all required dependencies are installed
  • Check network connectivity

Modal opens but looks broken?

  • Check for CSS conflicts
  • Verify AppKit version compatibility
  • Check theme configuration in appkit-config.ts

Browser Console Commands

Run these commands in the browser console to debug:

// Check if AppKit is enabled
console.log('AppKit Enabled:', process.env.NEXT_PUBLIC_USE_APPKIT);

// Check AppKit instance
console.log('AppKit Instance:', window.appKit);

// Check for initialization errors
console.log('Check console for AppKit errors');
